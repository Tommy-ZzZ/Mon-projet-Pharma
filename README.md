# 💊 Mon-projet-Pharma

**Gestion de clientèle et commandes de produits pharmaceutiques**  
Application multiplateforme : **Web (React)** et **Mobile (React Native / Expo)** pour gérer clients, commandes et produits pharmaceutiques.

---

## 🧠 Fonctionnalités principales

- 📋 Gestion des clients (ajout, modification, suppression)  
- 🛒 Prise de commandes de produits pharmaceutiques  
- 💻 Interface web interactive (React)  
- 📱 Application mobile cross-platform (React Native / Expo)  
- 📦 Suivi des produits et historique des commandes
